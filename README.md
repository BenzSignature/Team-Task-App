## Team Task App

A team-based task management Flutter app with real-time notifications when tasks are created or updated.

## ‚ú® Features

- ‚úÖ Create and manage team projects easily
- ‚úÖ Add team members to each project
- ‚úÖ Assign tasks to specific members
- ‚úÖ Each member can only see their own tasks
- ‚úÖ Get instant notifications when a task is added or updated
- ‚úÖ Store project and task data using Firestore
- ‚úÖ Send notifications with Firebase Cloud Messaging (FCM)
- ‚úÖ Reactive state updates using GetX
- ‚úÖ Clean project structure (MVC + Bindings)
- ‚úÖ Have custom widgets for reusable UI parts
- ‚úÖ Ready to support calendar view and task filters in the future

## üõ† Tech Stack

- Flutter 3.8.1  
- GetX (state management + routing)  
- Firebase Core & Cloud Firestore  
- Firebase Cloud Messaging (FCM)  
- Flutter Local Notifications
